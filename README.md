  ## About RideWave
  RideWave is a fast-growing ride-hailing and logistics startup
  operating across 6 cities in India. This project builds their
  complete data platform from raw CSV files to business-ready
  Gold tables using AWS and Databricks.

  ## Architecture
  AWS Track  : S3 → Glue → Athena → Redshift
  DB Track   : S3 → Auto Loader → Bronze → Silver → Gold → Workflow

  ## Tech Stack
  Python, SQL, PySpark, AWS S3, Glue, Athena, Kinesis,
  Redshift Serverless, Databricks, Delta Lake, GitHub
